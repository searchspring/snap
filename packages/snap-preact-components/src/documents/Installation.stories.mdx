# Installation

Install the package:
```bash
npm install @searchspring/snap-preact-components
```

Import component:
```js
import { facet } from '@searchspring/snap-preact-components';
```

Use component:
```jsx
<Facet facet={facet} optionsLimitCount={12} />
```
