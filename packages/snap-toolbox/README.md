# Snap Toolbox


A collection of utility tools such as DOM targeting, currency formatting & browser feature flags.

## Installation

```bash
npm install --save @searchspring/snap-toolbox
```


## Import
```js
import { 
    filters, 
    DomTargeter, 
    getFlags, 
    featureFlags, 
    cookies, 
    url,
    charsParams,
    until,
} from '@searchspring/snap-toolbox';
```
